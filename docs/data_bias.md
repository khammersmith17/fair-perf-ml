# Data Bias

Pre-training bias analysis. Import from `fair_perf_ml.data_bias`.

Supported metrics: Class Imbalance (CI), Difference in Proportion of Labels (DPL),
KL Divergence, JS Divergence, Lp-Norm, Total Variation Distance (TVD),
Kolmogorov–Smirnov (KS).
